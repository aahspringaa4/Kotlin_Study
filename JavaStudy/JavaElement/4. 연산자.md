## 논리 연산자

#### 조건식1 && 조건식2

조건식1과 조건식2가 모두 참일때 참 (And 게이트)

#### 조건식1 || 조건식2

조건식1과 조건식2중 하나라도 참이면 참 (Or 게이트)



## 삼항 연산자

조건식 ? 값 또는 연산식1 : 값 또는 연산식2

조건식이 참이면 값 또는 연산식1 실행

조건식이 거짓이면 값 또는 연산식 2 실행



```
int score = 95;
char grade = (score > 90) ? 'A' : 'B'
```

와

```
int score = 95
char grade;
if(score > 90){
   grade = 'A';
} else {
   grade = 'B'
}
```

는 같은 코드이다.
